StarPlot - Kyle Connor
Knox College

The purpose of this program is to illustrate the evolution of stars on an H-R diagram. 

Minimum shippable product features:
Contains an H-R diagram with logarythmic axies. When the diagram is clicked, a star is placed on it. 
From there, the star moves about the H-R diagram in a realistic manner. 

Additional Features
Be able to create a random population of stars/protostars.
Place stars from an index of real stars. 
Attractive interface.
